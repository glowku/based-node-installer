System-Specific Instructions:

Linux Instructions:

Download the script first
terminal
$ curl -sSL https://raw.githubusercontent.com/glowku/based-node-installer/main/install.sh -o install.sh
Convert Windows line endings to Linux
terminal
$ sudo sed -i 's/\r$//' install.sh
Make the script executable
terminal
$ sudo chmod +x install.sh
Run it with your parameters
terminal
$ ./install.sh "WALLET_ADDRESS" "NODE_NAME" "STAKE_AMOUNT" "SERVER_TYPE" "OS"

After Installation:

terminal
$ sudo systemctl status basedai
Check node status
terminal
$ sudo journalctl -u basedai -f
View node logs
terminal
$ sudo systemctl restart basedai
Restart node

Windows (WSL) Instructions:

Open PowerShell as Administrator
Download the script first
powershell
PS> Invoke-WebRequest -Uri "https://raw.githubusercontent.com/glowku/based-node-installer/main/install.sh" -OutFile "install.sh"
Convert Windows line endings to Linux
powershell
PS> (Get-Content install.sh) | Set-Content -NoNewline install.sh
Launch WSL
powershell
PS> wsl
Run it in WSL
wsl
$ ./install.sh "WALLET_ADDRESS" "NODE_NAME" "STAKE_AMOUNT" "SERVER_TYPE" "OS"

After Installation:

wsl
$ sudo systemctl status basedai
Check node status
wsl
$ sudo journalctl -u basedai -f
View node logs

macOS Instructions:

Open Terminal
Install Homebrew if not already installed
terminal
$ /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
Download the script first
terminal
$ curl -sSL https://raw.githubusercontent.com/glowku/based-node-installer/main/install.sh -o install.sh
Convert Windows line endings to Linux
terminal
$ sed -i 's/\r$//' install.sh
Make the script executable
terminal
$ chmod +x install.sh
Run it with your parameters
terminal
$ ./install.sh "WALLET_ADDRESS" "NODE_NAME" "STAKE_AMOUNT" "SERVER_TYPE" "OS"

After Installation:

terminal
$ sudo launchctl list | grep basedai
Check node status
terminal
$ tail -f /opt/basedai/logs/basedai.log
View node logs

General Installation Steps:

  1. Connect to your server via SSH
  2. Ensure you have sudo/administrator privileges
  3. Paste and run the command above
  4. Follow the on-screen instructions
  5. Your node will be automatically configured and started
